# SOC Analyst Lab Series

A hands-on SOC Analyst project series demonstrating practical cybersecurity skills in threat detection, incident response, network analysis, vulnerability assessment, and SIEM monitoring.

## Labs Covered

* Lab 1: Log Analysis & Threat Detection
* Lab 2: Incident Response Walkthrough
* Lab 3: Network Traffic Analysis (Wireshark)
* Lab 4: Vulnerability Scanning & Risk Analysis
* Lab 5: SIEM Alert Tuning
* Lab 6: Malware & Endpoint Analysis

## Tools Used

* Splunk
* Wireshark
* Nessus
* Nmap
* VirusTotal
* Linux
